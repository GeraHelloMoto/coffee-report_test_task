# Coffee Report Generator

Скрипт для генерации отчётов по данным о подготовке студентов к экзаменам.  
Обрабатывает один или несколько CSV-файлов, формирует отчёт в виде таблицы и выводит его в консоль.

## Пример запуска

```bash
python coffee_report.py --files data/students_2024.csv data/students_2025.csv --report median-coffee
```
